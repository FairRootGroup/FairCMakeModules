*****
Usage
*****

In your CMake-based project, discover this package with
:cmake:command:`find_package`

.. code-block:: cmake

  find_package(FairCMakeModules REQUIRED)

Then :cmake:command:`include` any module you would like to use

.. code-block:: cmake

  include(<modulename>)

Refer to each :ref:`module's documentation <modules_list>` on how to use it.
