FairSummary¶

New in version 0.2.0.

Requires CMake 3.12 or later.

Depends on FairFormattedOutput (and includes it automatically).

fair_summary_global_cxx_flags_standard()¶

fair_summary_global_cxx_flags_standard()

If CMAKE_CXX_FLAGS, or CMAKE_CXX_STANDARD are set, their values are displayed in a nice tabular way.

fair_summary_build_types()¶

fair_summary_build_types()

Display the available Build Types from CMAKE_CONFIGURATION_TYPES in a nice tabular way and mark the type, that the user has chosen.

fair_summary_package_dependencies()¶

fair_summary_package_dependencies()

find_package2() keeps track of all the found packages. This function displays them (including found version, requested version, and install prefix) in a nice tabulary way.

The prefix column prints the value of <pkgname>_PREFIX.
